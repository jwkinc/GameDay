GameDay Live v11 - Firebase Ready

How to use two devices:
1. Create a Firebase project.
2. Create a Realtime Database.
3. Copy your Firebase web app config.
4. Open index.html, go to Controller, paste the config into Two-Device Sync.
5. Pick a Game Room Name.
6. Tap Save Sync Settings, then Connect Sync.
7. Open the same app on the scoreboard device with the same room name and config, then Connect Sync.
8. One device can use Scoreboard mode, the other Controller mode.

Note:
YouTube links sync across devices.
Uploaded local video/audio files stay on the device where you selected them.
